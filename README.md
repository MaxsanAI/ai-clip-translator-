# AI Video Translator

A simple AI tool that translates videos into multiple languages.

Features:
- Upload videos
- Generate subtitles (SRT)
- Translate subtitles
- AI dubbing
- Burn-in subtitles
- Export ready videos for social media

Free version:
- max 3 minute video
- 2 videos per day
